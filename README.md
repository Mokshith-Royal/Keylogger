# Keylogger (Educational Demonstration)

This project is a consent-based educational demonstration of keyboard event monitoring developed for internship review purposes.

## Features
- Real-time key event monitoring
- Timestamped logging
- Start/Stop controls
- Log file generation
- Simple Tkinter GUI

## Technologies
- Python 3
- Tkinter
- pynput

## How to Run

```bash
pip install pynput
python app.py
```
